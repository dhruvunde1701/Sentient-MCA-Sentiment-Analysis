CORE FEATURE SET
=============================================================================

1. DUAL-MODEL SENTIMENT ANALYSIS
   - VADER Sentiment Analyzer: Rule-based analyzer optimized for social media text
   - TextBlob Analyzer: Pattern-based sentiment analysis with polarity scoring
   - Consensus Methodology: Intelligent resolution when models disagree
   - Classification Thresholds: Positive (>0.05), Negative (<-0.05), Neutral (between)
   - Individual Comment Analysis: Detailed sentiment scores for each comment
   - Overall Distribution: Aggregated sentiment percentages across all comments
   - Stakeholder Segmentation: Sentiment breakdown by entity type (Corporate, Individual, etc.)

2. ADVANCED TEXT SUMMARIZATION
   - Extractive Summarization: Uses LSA (Latent Semantic Analysis) and Luhn algorithms
   - Multi-Level Summaries: Overall, sentiment-based, and stakeholder-based groupings
   - Key Phrase Extraction: Frequency-based approach with consultation-specific filtering
   - Theme Identification: Automated extraction of main discussion topics
   - Fallback Mechanisms: Multiple summarization approaches for reliability
   - Content Preservation: Maintains original meaning through extractive methods

3. COMPREHENSIVE WORD CLOUD VISUALIZATION
   - Overall Word Clouds: Complete consultation vocabulary overview
   - Sentiment-Specific Clouds: Separate visualizations for positive, negative, neutral
   - Stakeholder Clouds: Word usage patterns by entity type
   - Color-Coded Analysis: Green (positive), Red (negative), Blue (neutral)
   - Frequency Analysis: Statistical breakdowns with interactive bar charts
   - Custom Stop Word Filtering: Consultation-specific term removal
   - Export Capabilities: High-resolution image generation

4. INTERACTIVE WEB INTERFACE
   - Streamlit-Based Application: Professional government-appropriate design
   - File Upload System: CSV format with comprehensive validation
   - Real-Time Processing: Live analysis with progress indicators
   - Tabbed Organization: Sentiment Analysis, Text Summaries, Detailed Results, Download
   - Progressive Disclosure: Prevents information overload through organized layout
   - User Action Logging: Detailed tracking of user interactions for debugging
